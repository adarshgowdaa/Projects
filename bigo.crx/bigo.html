Time Complexity

  • Binary Search :

    O(log)

  • Stack & Queue :

    O(1)

  • Sorting :

    O(nlog(n))

  • Build Binary Search Tree :

    O(n)

  • Search in Binary Search Tree :

    O(logn)

  • Build a Heap :

    O(n)

  • Push to Linked List :

    O(n)

  • Search in Linked List :

    O(n)

  • Insert/Search in Hashmap :

    O(1)
